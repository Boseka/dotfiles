# dotfiles.git
***

### Introduction
This repo contains mostly my own customized config files, including, but not limited to:
 - Shells
    + `zsh`
    + `bash`
 - editors
    + `vim`
    + `emacs`
 - other tools and configs
    + `screen`
    + `irssi`
    + `git`
    + `ncmpcpp`, `mpd`
    + `xbindkeys`
        + `xfce4-terminal`
    + `uzbl-browser`
